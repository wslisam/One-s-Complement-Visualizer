This interactive tool is designed to help students understand how computers store negative numbers using the **One's Complement** method. 

## 🎯 What will you learn?
Before computers settled on the modern method (Two's Complement), they experimented with One's Complement. It’s a very simple idea, but it has one massive flaw that you will discover by playing with this tool!

## 🚀 How to use this tool?
1. **Type a Number**: Enter any number between `-127` and `127` in the Decimal box. Watch how the 8-bit binary representation changes instantly.
2. **Click "+1" or "-1"**: See how adding or subtracting affects the binary bits step-by-step.
3. **Invert Bits (NOT)**: This is the magic button! Click it to flip all `0`s to `1`s, and all `1`s to `0`s. This is exactly how One's Complement creates a negative number.
4. **Show -0**: Wait, negative zero? Yes! Click this button to see the biggest flaw of One's Complement.

## 🧠 Check your understanding
After playing with the tool, scroll down to read the **Concept Breakdown** and **Common Misconceptions** sections. If you can explain to your classmate *why* modern computers stopped using One's Complement, you've mastered this topic!
